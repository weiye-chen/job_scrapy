# job_scrapy
### 概述
boss直聘岗位数据获取、清洗、入库和可视化。

boss直聘是所有招聘类网站中反爬最严格的网站，因此通过端口列表监听来获取接口数据。
### 使用说明
1. 安装所需包

```commandline
pip install DrissionPage pandas streamlit plotly
```

2. 配置文件
`config.json`
`queries`是岗位关键词列表，`cities`是城市代码列表，`delay_between_tasks`: 切换不同岗位/城市之间的等待时间，防止被封

```commandline

```

